Objects in JavaScript

You can shorthand objects with methods which is pretty cool.

user = { sayHi() { alert("Hello"); } };

Then you can call it: user.sayHi();

"This"

Using "this" in code is a really good way to be able to use specific information in an object for methods.

I've mostly used it in calling JavaScript functions that take an arguement, and that arguement is some input that the user will put in.

Something cool about using this in JavaScript is that it is unbound.
It can be used in any function.

Arrow functions are special: they don’t have their “own” this. If we reference this from such a function, it’s taken from the outer “normal” function.

Events

Events are things like clicks or changes to something.

There are things called Event Listeners that monitor and wait for events to happen

Event Listeners are used to trigger something when specific things happen that the developers want to keep an eye out for.

Many common Event Listeners are placed on buttons, so that they do something specific when clicked. They can save information, or even do math!

These are inline elments you can add like onclick="clicked()" and will call the function clicked() when the button gets clicked.
